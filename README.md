# SillyTavern Persona Generator

Generate vivid, roleplay-ready **SillyTavern  Personas** with AI.

This is a prompt/instruction-based generator for creating the persona that represents `{{user}}` in SillyTavern. Instead of generating a full Character Card for `{{char}}`, it creates a concise, playable, emotionally grounded **Persona Description** that can be pasted into SillyTavern's Persona system.

---

## How to Use

1. Copy or upload the instruction file to your preferred AI assistant.
2. Type something like:

```text
Execute these instructions and create a SillyTavern User Persona.
```

3. Answer the assistant's 2-3 clarifying questions.
4. Copy the generated Persona Description.
5. Open SillyTavern's Persona Management panel.
6. Create or select a persona.
7. Paste the generated text into the Persona Description field.
8. Select that persona before starting or continuing a chat.

That's it. The output is designed to be ready for direct use in SillyTavern.
